# NumPy & Pandas Lab

This repository contains my practice code for NumPy and Pandas, written while learning data analysis fundamentals in Python.

The scripts cover basic to intermediate concepts including array creation, indexing, slicing, DataFrame operations, filtering, and groupby operations. Some of this code was written a few months back and is being organized and pushed here now.

This is mainly for my own learning record and reference, and to track my progress in data analysis.

## Structure

- numpy/ - NumPy practice scripts (arrays, indexing, broadcasting, etc.)
- pandas/ - Pandas practice scripts (Series, DataFrames, filtering, groupby, etc.)

## Note

These are practice files, not production code. There may be mistakes or rough edges, but the goal is consistent learning and improvement over time.
